# **比特币的法律地位怎么样？**

根据我国《民法典》第一百二十七条的明确规定，法律对数据网络虚拟财产给予保护。这意味着你的比特币在法律层面是被认可和保护的。

早在2013年，央行等五部委发布的通知就明确表示：**比特币是一种特定的虚拟商品**。注意「商品」这两个字，既然是商品，你买来放在钱包里，就像买了一双限量版球鞋或者游戏皮肤一样，属于你的私人财产。

近年来，上海高院、北京法院等多地法院在实际判决中也多次重申：比特币具有价值性、稀缺性和可支配性，应当认定为虚拟财产，受法律保护。我记得看过好几个案例，法院都是按照虚拟财产来处理比特币纠纷的。

# **国家打击的是什么行为？**

很多人搞混了，以为国家禁止一切与比特币相关的活动。实际上，国家重点打击的是：

1. 利用比特币进行洗钱、诈骗等犯罪活动
2. 未经许可开设加密货币交易场所
3. 为境内居民提供虚拟货币交易撮合服务
4. 利用虚拟货币进行非法集资

只要你不参与这些非法金融业务，**单纯的个人投资和持有是完全没有问题的**。就像你可以收藏邮票、古董一样，收藏比特币作为投资也是个人自由。
